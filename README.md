# Board-Project

이 프로젝트는 Next.js를 이용한 게시판 만들기 토이 프로젝트로, 팀원들이 같은 주제로 **솔로 프로젝트**를 진행하며 서로 코드리뷰를 하는 방식으로 진행되었습니다.

진행 기간: 2023년 5월 24일 ~ 2023년 6월 9일(2주)
